# 三书会通

以袁枢《通鉴纪事本末》为纲，司马光《资治通鉴》为据，王夫之《读通鉴论》为鉴，三书合一，交互阅读。

## 内容

覆盖从战国到五代（前403年—960年）的 **43个核心历史事件**，按《通鉴纪事本末》42卷大纲展开：

| 时代 | 章节数 | 时间跨度 |
|------|--------|----------|
| 战国·秦 | 7 | 前403—前207 |
| 两汉 | 9 | 前206—220 |
| 三国 | 5 | 190—280 |
| 两晋 | 4 | 265—420 |
| 南北朝 | 4 | 420—589 |
| 隋 | 3 | 581—618 |
| 唐 | 8 | 618—907 |
| 五代 | 3 | 907—960 |

每个事件包含：
- **纪事本末** — 事件始末完整叙事
- **通鉴原文** — 对应卷数的原文引用
- **王夫之论** — 《读通鉴论》中的史论观点

## 功能

- 🌓 深色/浅色主题切换（奶白色高级色系）
- 📑 侧边栏导航 + 朝代快捷跳转
- 🔍 全文搜索
- 📱 响应式布局
- 🏷 时间/地点/人物/事件标签
- 📜 阅读进度条

## 使用

直接浏览器打开 `通鉴三书会通.html` 即可，无需服务器。
